static List< Card > domain.game.DealerService.shuffle ( List< Card > cards)
static

Baraja una lista de cartas utilizando el algoritmo moderno de Fisher-Yates.

El método crea una nueva lista de cartas barajadas sin modificar la lista original. Se utiliza SecureRandom para generar índices aleatorios de forma segura.

Parámetros
cardsLista de cartas a barajar. No se modifica.
Devuelve
Una nueva lista de cartas barajadas.
