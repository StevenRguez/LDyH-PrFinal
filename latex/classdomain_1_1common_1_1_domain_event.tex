\doxysubsection{Descripción detallada}
Clase abstracta base que representa un evento de dominio genérico en la aplicación. 

Un evento de dominio captura una ocurrencia específica dentro del dominio, incluyendo una marca de tiempo de cuándo ocurrió. Las subclases deben heredar de \doxyref{Domain\+Event}{pág.}{classdomain_1_1common_1_1_domain_event} para representar tipos específicos de eventos.
