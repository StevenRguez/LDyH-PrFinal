\doxylink{classapplication_1_1dto_1_1_player_info_d_t_o}{Player\+Info\+DTO} Objeto de Transferencia de Datos (DTO) que contiene la información del jugador para la transferencia de datos entre capas.

Esta clase representa la información esencial de un jugador, encapsulando su identificador único (UUID) y su nombre. Implementa Serializable para permitir la serialización del objeto para la transferencia de datos.
