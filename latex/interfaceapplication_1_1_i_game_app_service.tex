\doxysubsection{Descripción detallada}
\doxylink{interfaceapplication_1_1_i_game_app_service}{IGame\+App\+Service} Interfaz que define el servicio de aplicación para interactuar con una sesión de juego.

Proporciona métodos para recuperar información de los jugadores, gestionar acciones de los mismos y consultar el estado del juego. Este servicio está diseñado para separar la capa de aplicación de la capa de dominio.
