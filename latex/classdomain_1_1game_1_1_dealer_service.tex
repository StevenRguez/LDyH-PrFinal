Baraja una lista de cartas utilizando el algoritmo moderno de Fisher-\/\+Yates. 

El método crea una nueva lista de cartas barajadas sin modificar la lista original. Se utiliza {\ttfamily Secure\+Random} para generar índices aleatorios de forma segura.
